# Lycée Privé International Berthe & Jean

Site vitrine Next.js du Lycée Privé International Berthe & Jean à Essassa.

## Démarrer en local

```bash
npm install
npm run dev
```

Ouvrir ensuite `http://localhost:3000`.

## Commandes

```bash
npm run lint
npm run build
```

## Contenu

- Page d'accueil statique en français
- Logo officiel du lycée
- Sections: accueil, à propos, programmes, admissions, vie scolaire, actualités et contact
- Images du campus et de la vie scolaire dans `public/assets`
